## Features

- Add / Remove Movies to your collection
- Favorite / Unfavorite movies

## Run Locally

Clone the project

```bash
  git clone https://github.com/sauravshekharsingh/movie-app.git
```

Go to the project directory

```bash
  cd movie-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Environment Variables

'REACT_APP_API_KEY'
